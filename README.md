## 📘 About

This portfolio website is a comprehensive overview of who I am as a developer — showcasing my skills in software development, web technologies, and problem-solving, along with my work experiences, projects, and achievements. It includes a brief introduction about me, links to my social media handles and coding profiles such as GitHub, LeetCode, and Codeforces, as well as access to my resume for professional reference. Whether you're a recruiter, collaborator, or fellow developer, this site provides a complete snapshot of my technical journey and professional presence.

## ⚙️ Installation

# Clone the repo

```bash
git clone https://github.com/ashutosh229/portfolio-next.git
```

# Navigate into the directory

```bash
cd portfolio-next
```

# Install dependencies

```bash
npm install
```

# Start the server

```bash
npm run dev
```
